# YoutubeMusicAutoConfirm

Automatically comfirms the "I understand and wish to proceed" button for you on Youtube Music, stopping the interruptions caused by music Youtube deems to be explicit and require listener discretion.
This extension works even while the tab is unfocused.

# Disclaimer

This extension is by no means a replacement for any such discretion while listening to explicit music and by using this extension you are confirming that you understand this and do not need any viewer discretion warnings.
You are responsible for any potentially distressing music you choose to listen to while using this extension.

This extension does NOT automatically confirm the automatic pauses caused by idling while listening. There is already a great extension made for this which can be found here: https://github.com/lawfx/YoutubeNonStop
